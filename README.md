# Calculator

Input format: operators and operands sre separated by a space.

Example: 

```javascript
const a = new Calculator();
a.calculate('1 + 3 - 3 * 5');
```


# Installing
Run in console:

   `npm i`

# Run tests
- Tests without coverage

    `npm test`

- Tests with coverage

    `npm run test-with-coverage`
    
    Coverage info: `/coverage/lcov-report/index.html`
